## 关于 k3ay0

前端开发者，对 Web 安全和人工智能领域保持持续关注。相信技术的价值在于解决问题，而不仅仅是使用工具。

探索前端边界，守护安全底线，思考 AI 未来 —— 这三件事构成了我目前的技术主线。

## 技术方向

### 前端开发

日常工作的核心领域。从 Vue 到 React，从 JavaScript 到 TypeScript，从 Webpack 到 Vite，前端生态的快速迭代要求保持学习的节奏。关注组件化架构设计、工程化实践和性能优化，追求代码的可维护性与用户体验的平衡。

### Web 安全

安全不是附加项，而是开发的基本素养。从 XSS、CSRF 到 SQL 注入，从渗透测试到漏洞分析，理解攻击手段是为了更好地防御。关注 OWASP Top 10，也关注云原生环境下的安全架构设计。

### 人工智能

大模型时代正在重塑技术格局。从机器学习基础到 LLM 应用，从提示词工程到 RAG 系统，从模型安全到 AI 红队攻防，这个领域既有深度也有广度。保持对新技术的敏感度，同时警惕安全与隐私风险。

## 兴趣爱好

- **阅读** — 技术书籍和安全领域的研究论文，偶尔也读一些非技术类的书换换脑子

- **写作** — 通过博客记录学习过程，把零散的知识整理成体系
